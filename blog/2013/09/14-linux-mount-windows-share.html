Linux 掛載 Windows 網芳

2013-09-14 14:41

安裝

apt-get install smbclient
apt-get install cifs-utils

瀏覽

smbclient -L //IP/Share -U domain/username -D subdirectory

掛載

mount -t cifs -o user=<username>,domain=<domain>,password=<password> //IP/Share/subdirectory /mount_point
